# Adding Comments to Tickets

To add a comment to tickets, create a script that does just that. The arguments are passed in the following order:

1. `$ticket = $args[0]`

1. `$comment = $args[1]`

1. `$user = $args[2]`

1. `$password = $args[3]`

1. `$url = $args[4]`
